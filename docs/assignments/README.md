# Atividades do Livro

Este diretório contém as atividades práticas que acompanham o conteúdo do livro.

## Organização

As atividades estão organizadas de forma a complementar os capítulos do livro:

- Cada atividade está associada a **um ou mais capítulos**.
- Cada capítulo pode possuir **uma ou mais atividades relacionadas**.

Essa estrutura permite reforçar conceitos sob diferentes perspectivas e níveis de complexidade.

## Versionamento por Período

As atividades são **atualizadas e/ou geradas a cada semestre/ano**, podendo sofrer ajustes para:

- refletir melhorias pedagógicas
- incorporar novos exemplos e tecnologias
- adaptar o nível de dificuldade conforme o perfil da turma

## Objetivo

O objetivo das atividades é:

- consolidar o aprendizado teórico
- estimular a prática de desenvolvimento
- promover o raciocínio crítico e a resolução de problemas

## Observação

Recomenda-se que as atividades sejam realizadas **em paralelo ao estudo dos capítulos correspondentes**, respeitando a progressão sugerida no livro.
